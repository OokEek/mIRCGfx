# mIRCGfx
DLL for use with mIRC v7.83+ /drawdll command.

# Description
The dll provides two functions that can be called from MSL these are DllDraw and DllDrawCopyNext

DllDrawCopyNext
  Only called internally by the dll.
  
DllDraw
  Only called by the mIRC /drawdll command.
  See the supplied demo script for usage.
